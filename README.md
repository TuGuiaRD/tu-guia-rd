# Tu Guia RD

Primera beta estatica de una web de calculadoras y guias practicas para Republica Dominicana.

## Archivos principales

- `index.html`: estructura de la web.
- `styles.css`: diseno visual responsive.
- `app.js`: calculadoras y comportamiento de la interfaz.
- `contenido-inicial.md`: plan inicial de contenido y crecimiento.

## Como verla

Abre `index.html` en el navegador.

## Proximos pasos recomendados

1. Revisar el nombre, colores y secciones.
2. Convertir las guias principales en paginas individuales.
3. Publicar en Vercel, Netlify o Cloudflare Pages.
4. Crear Google Search Console y enviar el sitemap.
5. Crear Google Analytics.
6. Agregar politica de privacidad completa.
7. Publicar mas contenido antes de solicitar AdSense.
